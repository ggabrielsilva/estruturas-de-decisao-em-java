# Estruturas de Decisão em Java

Este repositório contém programas simples em Java utilizando estruturas de decisão (`if/else`).

## 📌 Programas

### 🔹 Ex161 - Número Par ou Ímpar

O programa recebe um número inteiro e:

* Verifica se é par ou ímpar
* Se for par, mostra o quadrado
* Se for ímpar, mostra o cubo

---

### 🔹 Ex162 - Sistema de Segurança de Elevador

O programa simula um sistema de controle de peso:

* Recebe o peso de duas caixas
* Soma os pesos
* Verifica se ultrapassa o limite permitido
* Exibe se a operação é segura ou se há sobrecarga

---

## 💻 Tecnologias

* Java
* Scanner (entrada de dados)

---

## 🎯 Objetivo

Praticar lógica de programação e estruturas de decisão em Java.
